
## סשן תיקוני ZozoBet + ביקורת Swarm (20/02/2026)

### תיקוני BettingPlatform שבוצעו:
1. **DC odds תואמים bet365** — הועתק parseEventOdds + syncLiveEventOdds מ-sandbox, סונן esports, הוגדל limit ל-15
2. **Backend לא דורס live odds** — events.js full-odds endpoint שומר על 6 live markets
3. **Odds validation** — bets.js מוודא שהיחסים תואמים DB (5% tolerance), מחזיר 409 odds_changed
4. **זמן משחק מתעדכן** — full-odds endpoint מחזיר matchMinute/matchPeriod/timer/scores + renderMM מעדכן header
5. **visibilitychange** — מרענן odds + timer כשחוזרים ללשונית
6. **isLive fix** — לא מסמן משחקים כלייב לפני שהתחילו
7. **Sandbox synced** מ-production

### ביקורת Swarm (Claude Code):
- **כל 7 בוטים עובדים** (or, shomer, koder, tzayar, worker, researcher, bodek)
- **send.sh עובד** עם 5 באגים: JSON injection, no token check, jq crash, caption limit, mixed content-type
- **56 שיעורים** במערכת הלמידה
- **sessions_send לא אמין** — לפעמים agents לא מתעוררים
- **Claude Code CLI מותקן** (v2.1.44) ועובד! אבל נהרג (SIGKILL) על משימות ארוכות — צריך חקירה
- **דו"ח מלא:** swarm/AUDIT-REPORT.md

### החלטות:
- **עוברים ל-Claude Code CLI** לכתיבת קוד במקום sub-agents רגילים
- **יוסי רוצה פרויקט נפרד** לבדיקת מערכת הסוכנים — לא לגעת בבטים
- **חוק ברזל:** אור לא כותב קוד בעצמו, תמיד דרך Claude Code / סוכנים, sandbox בלבד
- **צריך לחקור:** למה Claude Code נהרג על משימות ארוכות (SIGKILL, לא OOM)
