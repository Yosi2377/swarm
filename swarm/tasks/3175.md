## בעיה

ב-DB יש 311 אירועים ב-24 שעות הקרובות.

## סיבה
ב-`events.js` שורה ~130 הquery מגביל ל-3 שעות:
```javascript
commenceTime: { $gte: now, $lte: new Date(now.getTime() + 3*60*60*1000) }
```

## מה צריך
1. שנה את החלון מ-3h ל-**24h** בדף הראשי
2. **לייב רץ** (live.html) ישאר רק live events (כמו שהוא)
3. גם: שנה לגישת **blacklist** במקום whitelist — חסום רק esoccer/setka/dota/ספאם, העבר הכל שאר

### שינוי ב-events.js:
```javascript
// BEFORE
const h3 = new Date(now.getTime() + 3*60*60*1000);
// AFTER
```

### שינוי isAllowedSport — blacklist approach:
```javascript
const SPORT_BLOCK = /esoccer|e_soccer|battle|volta|tt_cup|tt_elite|setka|liga_pro|czech.liga.pro|h2h_gg|cs2|lol|dota|rugby|nfl|ncaa|handball|volleyball|six_nations|slam|kickoff|utr.pro|cyber|virtual|sim_|esport/i;

function isAllowedSport(sportKey, sportTitle) {
  if (!sportKey) return false;
  if (SPORT_BLOCK.test(sportKey)) return false;
  if (sportTitle && SPORT_BLOCK.test(sportTitle)) return false;
  if (/^(soccer|basketball|tennis)/.test(sportKey)) return true;
  return false;
}
```
