# Task 3609 — עיצוב סרגל כלים עליון (header nav)

## מצב נוכחי
הסרגל העליון (.hdr) מכיל טקסטים פשוטים: ספורט, ליין רץ, ההימורים שלי, תוצאות, סטטיסטיקה, ניהול, יציאה
נראה פשוט מדי — טקסט + אייקון בלי עיצוב כפתור.

## מטרה
כפתורי nav מעוצבים בסגנון bet365:
- רקע כהה עם gradient עדין
- border-radius (rounded)
- hover effect (glow / brighten)
- active state (tab נבחר בולט)
- padding + spacing מסודר
- transition animation חלקה

## דוגמה לסגנון bet365:
```css
.nav-btn {
  background: linear-gradient(180deg, #2d2d2d, #1a1a1a);
  border: 1px solid #444;
  border-radius: 6px;
  padding: 8px 16px;
  color: #e0e0e0;
  cursor: pointer;
  transition: all 0.2s;
  display: flex;
  align-items: center;
  gap: 6px;
  font-size: 14px;
}
.nav-btn:hover {
  background: linear-gradient(180deg, #3a3a3a, #2d2d2d);
  border-color: #ffca28;
  color: #fff;
}
.nav-btn.active {
  background: linear-gradient(180deg, #1b5e20, #004d40);
  border-color: #4caf50;
  color: #ffca28;
  box-shadow: 0 0 8px rgba(76,175,80,0.3);
}
```

## חשוב
- לא לשנות את הפונקציונליות (onclick, setTab, etc.)
- לא לשנות אייקונים/טקסט
- רק CSS + class changes
- לעצב גם את כפתור "יציאה" ו-"EN/HE" ו-"יתרה"
- Badge של ליין רץ (מספר אדום) חייב להישאר
