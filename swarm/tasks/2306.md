# Task 2306 — ביטול הימור מחזיר כסף ליתרה

## בעיה
כשמנהל מבטל הימור של שחקן (ב"הימורים שלי"), הכסף לא חוזר ליתרה של המשתמש.

## מה צריך
1. כשסטטוס הימור משתנה ל-"void" או "cancelled":
   - הסכום (stake) חוזר ליתרה של המשתמש
   - נוצרת טרנזקציה `bet_void` / `bet_cancel`
2. בדוק ב-bets.js — ה-PUT/PATCH endpoint שמעדכן הימורים
3. הוסף לוגיקה: if new status = void/cancelled → user.balance += bet.stake
4. **SELF-TEST**: צור הימור, בטל אותו, ראה שהיתרה חזרה!
