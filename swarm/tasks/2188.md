# Task 2188 — עיצוב מודאל הימורים כמו bet365 + זמני משחק

## מה צריך
1. **עיצוב מודאל הימורים כמו bet365** — כשנכנסים למשחק, האפשרויות צריכות להיראות כמו bet365 (ראה https://www.bet365.com/#/IP/EV151300098982C1)
   - Layout של שוקים (markets) בסגנון bet365: כותרות ברורות, odds בכפתורים
   - כל ה-markets שיש לנו צריכים לעבוד (לא רק tabs אלא גם הלחיצה על odds = הוספה לסליפ)
   - עיצוב נקי, מקצועי, RTL
   
2. **זמני משחק LIVE** — להציג בכל משחק חי:
   - דקה נוכחית (TM field מ-inplay)
   - תקופה/מחצית (1st Half, 2nd Half, HT, FT)
   - הזמן צריך להתעדכן בזמן אמת
