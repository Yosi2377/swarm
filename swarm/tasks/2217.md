# Task 2217 — זמני משחק מדויקים LIVE

## מה צריך
הוספת זמן משחק מדויק (דקה + תקופה) לכל משחק LIVE ב-ZozoBet.

## מקור הנתונים (BetsAPI inplay)
כל אירוע EV מכיל:
- **TM** = דקה נוכחית (e.g., 45, 89, 24)
- **TU** = timestamp של עדכון הטיימר (e.g., "20260215061034")
- **GO** = תקופה כטקסט ("1st", "3rd", "5th", "6th")

### מיפוי תקופות כדורגל:
- "1st" = מחצית ראשונה
- "3rd" = הפסקה (HT)
- "5th" = מחצית שנייה
- "6th" = HT של מחצית שנייה? (צריך לבדוק)

### כדורסל:
- רבעים (Q1-Q4) + Overtime

### טניס:
- סטים (Set 1, Set 2, etc.)

## מה לעדכן

### 1. Aggregator (`aggregator/src/inplay-parser.js`)
- לשמור TM (minute) ו-GO (period) בכל אירוע parsed
- להמיר GO לטקסט קריא (מחצית 1, הפסקה, מחצית 2, etc.)

### 2. Aggregator (`aggregator/src/index.js`)
- ב-syncLiveScores: לשמור minute + period ב-DB

### 3. Backend Events model + route
- להוסיף שדות: `matchMinute`, `matchPeriod` ב-Event schema
- להחזיר אותם ב-API response

### 4. Frontend (`index.html` + `live.html`)
- להציג ליד כל משחק LIVE: "45' | מחצית 2" או "Q3 | 24'"
- בטבלה הראשית + במודאל
