# Task 2374 — דאשבורד Backend + API תקין

## בעיה
הדאשבורד (https://tworkswarm.duckdns.org, port 8090) לא מציג נתונים אמיתיים.

## מה צריך לתקן

### APIs שצריכים לעבוד:
1. **GET /api/tasks** — משימות פעילות + היסטוריה
   - קרא מ-`swarm/tasks.json` (active+completed)
   - קרא מ-`swarm/tasks/*.md` (task files)
   - מיזוג: כל task file + status מ-tasks.json
   
2. **GET /api/scores** — ציוני סוכנים ✅ (כבר עובד)

3. **GET /api/lessons** — לקחים אחרונים
   - קרא מ-`swarm/learning/lessons.json`

4. **GET /api/agents** — סטטוס סוכנים בזמן אמת
   - קרא מ-`swarm/memory/shared/active-context.md`
   - או בדוק sessions פעילים

5. **GET /api/timeline** — ציר זמן של אירועים
   - git log אחרון
   - משימות שהושלמו
   - lessons חדשים

6. **SSE /api/stream** — עדכונים בזמן אמת
   - chokidar watches על tasks.json, tasks/, learning/

### עדכון tasks.json
- task.sh add/done כבר כותב ל-tasks.json
- הבעיה: רוב המשימות לא מוסיפות עם task.sh
- פתרון: הAPI יקרא גם מ-tasks/*.md files
