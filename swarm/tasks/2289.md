# Task 2289 — דוח רווח/הפסד + עמלות סוכנים

## מה צריך

### 1. פאנל ניהול — תצוגה היררכית עם רווח/הפסד
כל תפקיד רואה את המשתמשים שלו:
- **Admin** — רואה הכל (סוכני-על → סוכנים → שחקנים)
- **סוכן-על** — רואה את הסוכנים שלו + השחקנים של הסוכנים שלו
- **סוכן** — רואה את השחקנים שלו

### 2. לכל שחקן — דוח רווח/הפסד
- סה"כ הימורים (stake)
- סה"כ זכיות (wins)
- רווח/הפסד (wins - stake)
- **20% הנחה** — האתר נותן 20% הנחה לשחקנים מההפסד שלהם
  - אם שחקן הפסיד 1000₪, הוא מקבל 200₪ הנחה (20%)
  - מוצג כעמודה נפרדת

### 3. לכל סוכן/סוכן-על — סיכום + עמלה 10%
- סה"כ של כל השחקנים שלו (stake, wins, profit/loss)
- **10% עמלה** — הסוכן/סוכן-על צריך לשלם 10% מהרווח הנקי של השחקנים שלו לאתר
  - רווח נקי = סה"כ הפסד שחקנים - 20% הנחות ששולמו
  - 10% מזה = מה שהסוכן חייב לאתר
- מוצג בשורה נפרדת:
  - "סה"כ רווח נקי: X₪"
  - "10% עמלה לאתר: Y₪"
  - "סה"כ לשלם: Z₪"

### דוגמה:
```
סוכן: משה
├── שחקן יוסי: הימר 5000₪, זכה 3000₪ → הפסד 2000₪ → הנחה 20% = 400₪
├── שחקן אבי: הימר 3000₪, זכה 4000₪ → רווח 1000₪ → הנחה 0 (לא הפסיד)
├── שחקן דנה: הימר 2000₪, זכה 500₪ → הפסד 1500₪ → הנחה 20% = 300₪
│
├── סיכום שחקנים: הפסד כולל 2500₪, הנחות 700₪
├── רווח נקי (אחרי הנחות): 1800₪
├── 10% עמלה לאתר: 180₪
└── סה"כ לשלם: 180₪
```

### 4. תצוגה מופיעה ב:
- פאנל ניהול (admin.html) — admin רואה הכל
- פאנל סוכן-על (superagent.html) — רואה את שלו
- גם סוכן רגיל צריך לראות את השחקנים שלו
