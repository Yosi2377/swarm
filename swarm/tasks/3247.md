## בעיה 2: Cache על HTML
הוסף no-cache headers על index.html ו-live.html ב-backend.

### תיקון
ב-`backend/src/index.js` הוסף middleware:
```javascript
app.use((req, res, next) => {
  if (req.path.endsWith('.html') || req.path === '/') {
    res.set('Cache-Control', 'no-cache, no-store, must-revalidate');
    res.set('Pragma', 'no-cache');
    res.set('Expires', '0');
  }
  next();
});
```
