# Task 2375 — דאשבורד עיצוב UI מקצועי

## בעיה
ה-UI של הדאשבורד בסיסי מדי ולא מציג את המידע בצורה ברורה.

## מה צריך

### 1. Header
- לוגו TeamWork + שם
- שעון + תאריך
- כפתור refresh

### 2. כרטיסי סיכום (Summary Cards)
- משימות פעילות (מספר)
- משימות שהושלמו היום
- סוכנים פעילים עכשיו
- ציון ממוצע כללי

### 3. סוכנים — כרטיסים
- כרטיס לכל סוכן: אימוג'י + שם + תפקיד
- ציון (progress bar צבעוני)
- סטטוס: פעיל/לא פעיל (ירוק/אפור)
- משימה נוכחית (אם יש)
- tasks completed / total

### 4. משימות — טבלה/רשימה
- כל משימה: #ID, כותרת, סוכן, סטטוס, זמן
- צבע לפי סטטוס: ירוק=done, צהוב=active, אדום=stuck
- לחיצה → פתיחת נושא בטלגרם

### 5. לקחים — רשימה
- אחרונים למעלה
- צבע לפי severity: אדום=critical, צהוב=medium, כחול=low
- סוכן + תאריך

### 6. Timeline — ציר זמן
- אירועים אחרונים (git commits, tasks done, lessons)
- כרונולוגי

### 7. עיצוב כללי
- Dark theme (רקע כהה, טקסט בהיר)
- RTL תמיכה
- רספונסיבי (מובייל)
- Polling כל 5 שניות — ללא ריענון דף
