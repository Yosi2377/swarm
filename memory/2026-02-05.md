### Installation Method

Unity Hub GUI crashes on VPS without GPU. Solution: use headless CLI mode!

```bash
# Install xvfb first
apt install xvfb

# List available Unity versions
xvfb-run -a unityhub --no-sandbox -- --headless editors -r

# List installed editors
xvfb-run -a unityhub --no-sandbox -- --headless editors -i

# Install Unity with modules
xvfb-run -a unityhub --no-sandbox -- --headless install --version 6000.0.67f1 -m webgl
```

### Key Learnings
- Unity Hub GUI has GPU issues on VPS - use `--disable-gpu` flag or headless CLI
- Headless CLI bypasses all GPU problems
- Installation takes ~30 minutes for base editor + modules
